# UHU calendar using React js

A react calendar from scratch without any calendar library

## Running the Demo

1.  Clone the repo
2.  `npm install`
3.  `npm start`

## Uhu Calendar Features

1.  Add event
2.  Edit event
3.  Delete event
4.  Traverse between weeks
5.  Can add multiple event at the single time
